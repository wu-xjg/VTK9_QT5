# VTK9_QT5_VS2022
三维画图程序_vtk

支持画直线，圆弧，立方体
  （up 圆弧度数+10度
   down圆弧度数-10度）
支持两种读写保存方式
  按actor类别保存，按actor个数保存
支持撤销更改，清空
支持编辑对象
  选择actor删除
  （ 重载onKeyPress ）
  （backspace删除）
  （Ctrl+Z 撤销删除）
  平移，旋转（重载onKeyPress）
  （up 上移0.1,down下移0.1）
  （left左移0.1,right右移0.1）
  （1 左旋10度，2 右旋10度）
支持图形变换
  鼠标右键拾取对象
支持边提取，点提取
支持坐标轴指示
支持三种拾取方式和高亮显示
